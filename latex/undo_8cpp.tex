\subsection{Detailed Description}
Implementation of undo functions.

The undo system requires calling start\+Undo() when starting a G\+UI command and calling end\+Undo() when ending the command. All changes to a score in response to a G\+UI command must be undoable/redoable by executing a sequence of low-\/level undo operations. This sequence is built by the code handling the command, by calling one or more undo\+Op()\textquotesingle{}s between start\+Undo() and end\+Undo().
