67 /* How the fermatas are represented in Guitar Pro is two integers, the
68  * first is an index value and the second is the time division that
69  * index value refers to, and they are givin with respect to a
70  * measure. Time division 0 means a minim, 1 is a crotchet, 2 is a
71  * quaver and so on, with the index (counting from 0) refering to how
72  * many time divisions occur before the fermata. These numbers are
73  * separated in GP6 with a '/' character. For example, a note
74  * occurring on the third beat of a measure in a 4/4 bar would be
75  * represented as 2/1.
76  */
77 struct GPFermata {
78  int index;
79  int timeDivision;
80 };
