◆ drop()

Element * Ms::ChordRest::drop ( EditData &  )
overridevirtual

Handle a dropped element at canvas relative pos of given element type and subtype. Returns dropped element if any. The ownership of element in DropData is transfered to the called element (if not used, element has to be deleted). The returned element will be selected if not in note edit mode.

Reimplemented by elements that accept drops.
