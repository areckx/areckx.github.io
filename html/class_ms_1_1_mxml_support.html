static

Convert a string in s into an int. Set *ok to true iff conversion was successful. s may end with ".0", as is generated by Audiveris 3.2 and up, in elements <divisions>, <duration>, <alter> and <sound> attributes dynamics and tempo. In case of error val return a default value of 0. Note that non-integer values cannot be handled by mscore.
