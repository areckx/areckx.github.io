Detailed Description

An URL to browse.

The LinkBrowse link holds a URL (eg 'http://poppler.freedesktop.org', 'john@.nosp@m.some.nosp@m..org', etc) to be open.

The format of the URL is specified by RFC 2396 (http://www.ietf.org/rfc/rfc2396.txt)
