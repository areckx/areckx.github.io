25 // Mapping of PDFDocEncoding (used to represent text values such as document
26 // metadata or annotation text) to Unicode codepoints. Not all 8-bit values in
27 // PDFDocEncoding are defined; undefined bytes are mapped to U+FFFD (Unicode
28 // replacement character).
29 //
30 // PDFDocEncoding is only directly defined for printable characters, but some
31 // control characters such as carriage return will still be used. We define
32 // mappings of the standard whitespace control characters (tabs, newlines) to
33 // the corresponding Unicode values. Other control characters are left
34 // undefined.
